# E-Commerce Sales Dashboard

## Project Overview
Designed an interactive Excel dashboard to analyze sales performance across product categories, regions, and months.

## Business Problem
An E-Commerce company wanted actionable insights to improve business performance using historical sales data.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Combo Box Controls
- Data Analysis Add-in

## Dashboard Features
- Monthly Sales Trend
- Monthly Profit Trend
- Region-wise Sales Analysis
- Product Category Filter (Combo Box)
- Interactive Reporting

## Key Findings
- October is the highest-selling month.
- October is the most profitable month.
- Central region generates the highest sales.

- Data Cleaning
- Dashboard Design
